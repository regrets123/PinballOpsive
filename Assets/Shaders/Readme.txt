When you use Unity 2020 or above with HDRP/URP version 10.x you need update shader 10.x following this steps:
- Delete old Shader folder in package.
- Import Shader 10x including package.
You should following video tutorial how to update shader 10.x: https://www.youtube.com/watch?v=jtmrut6LlLE&t=5s
